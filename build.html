Build Guide

🔧 Prerequisites

  • OpenJDK
  • Python 3
  • aria2c
  • zip
  • lz4
  • make
  • mkdtimg
  • imjtool
  • erofs-utils
  • OpenSSL
  • xmlstarlet
  • Android NDK
Note: Missing any of these dependencies may result in build failures.

🛠️ Build or Modify the ROM

1️⃣ Configure src/makeconfigs.prop

  • Edit the file to match your needs.
  • → Configuration guide

2️⃣ Configure src/genericTargetPropeties.conf

  • Update this if your device isn't officially supported.
  • → View supported devices
  • → Configuration guide (only if not supported)

3️⃣ Build the ROM

  • Run the following command:
./src/build.sh <firmware_url> | <local_firmware_path>

If already configured, simply run:

./src/build.sh

4️⃣ Test the ROM

  • Flash and thoroughly test the ROM on your device.
Important: Ensure the firmware matches your device model.
