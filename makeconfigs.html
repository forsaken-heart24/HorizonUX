Build Configuration Variables | makeconfigs

Please note that while everything works, some configurations might cause issues. Some of these configs can lead to a bootloop.

Debugging

  • TARGET_BUILD_IS_FOR_DEBUGGING: Enables verbose logging for debugging, which impacts performance.

Security

  • TARGET_REMOVE_NONE_SECURITY_OPTION: Disables the "None" option for lock screen security.
  • TARGET_REMOVE_SWIPE_SECURITY_OPTION: Disables the swipe-to-unlock option.

Bloatware & Features

  • TARGET_REMOVE_USELESS_VENDOR_STUFFS: Removes bloat from the vendor partition.
  • TARGET_REMOVE_USELESS_SAMSUNG_APPLICATIONS_STUFFS: Removes unnecessary Samsung apps (note: Android 9 is not fully supported).
  • TARGET_ADD_EXTRA_ANIMATION_SCALES: Adds extra animation scales for customization.
  • TARGET_FLOATING_FEATURE_ENABLE_BLUR_EFFECTS: Enables live blur effects for aesthetics. Please note that this impacts battery life and performance.
  • TARGET_FLOATING_FEATURE_ENABLE_ULTRA_POWER_SAVING: Enables ultra power saving. This feature is generally unnecessary, but you can enable it if you want.
  • TARGET_FLOATING_FEATURE_DISABLE_SMART_SWITCH: Disables Smart Switch during setup.

Special Features

  • TARGET_FLOATING_FEATURE_ENABLE_ENHANCED_PROCESSING: Attempts to boost performance at the cost of increased heat and reduced battery life.
  • TARGET_INCLUDE_UNLIMITED_BACKUP: Enables unlimited photo backup in a specific app.
  • TARGET_INCLUDE_SAMSUNG_THEMING_MODULES: Installs patched Samsung Good Lock modules.
  • TARGET_FLOATING_FEATURE_INCLUDE_SPOTIFY_AS_ALARM: Includes Spotify as an option for alarm tones.
  • TARGET_FLOATING_FEATURE_INCLUDE_EASY_MODE: Enables Easy Mode, which provides larger icons and is designed for users with visual difficulties.
  • TARGET_FLOATING_FEATURE_INCLUDE_CLOCK_LIVE_ICON: Enables a live clock icon in the launcher. Disable this if you prefer.
  • TARGET_FLOATING_FEATURE_ENABLE_EXTRA_SCREEN_MODES: Requires proper or minimum mdNIE support in the ROM and possibly the device for proper functionality.
  • TARGET_FLOATING_FEATURE_ENABLE_VOICE_MEMO_ON_NOTES: Enables the voice memo feature in the Notes app. Only supported on UI7.
  • TARGET_BUILD_ENABLE_SEARCLE: Enables Searcle (Circle to search) feature. OneUI 6 and Above.

Audio & Display

  • TARGET_INCLUDE_HORIZON_AUDIO_RESAMPLER: Fixes LDAC audio distortion for lower-end Bluetooth audio devices.
  • TARGET_FLOATING_FEATURE_SUPPORTS_DOLBY_IN_GAMES: Enables Dolby audio in games (only if supported and if the hardware is capable).

Additional Customization

  • TARGET_INCLUDE_HORIZON_AUDIO_RESAMPLER: Fixes LDAC audio distortion for lower-end Bluetooth audio devices.
  • TARGET_FLOATING_FEATURE_SUPPORTS_DOLBY_IN_GAMES: Enables Dolby audio in games (only if supported and if the hardware is capable).
  • TARGET_FLOATING_FEATURE_LAUNCHER_CONFIG_ANIMATION_TYPE: Adjusts launcher animation for different performance tiers (LowEnd, LowestEnd, LowEndFast, Mass (mid-rangers, not available after 3.1), HighEnd, CHNHighEND, and HighEnd_Tablet).
  • CUSTOM_WALLPAPER_RES_JSON_GENERATOR: Adds multiple wallpapers easily.
  • TARGET_BUILD_ADD_MOBILE_DATA_TOGGLE_IN_POWER_MENU: Adds a mobile data toggle to the power menu.
  • TARGET_BUILD_FORCE_FIVE_BAR_NETICON: Forces the network icon to display 5 bars.
  • TARGET_BUILD_ADD_CALL_RECORDING_IN_SAMSUNG_DIALER: Enables call recording in the Samsung Dialer app. Note: You are responsible for complying with local laws.
  • TARGET_BUILD_FORCE_SYSTEM_TO_NOT_CLOSE_CAMERA_WHILE_CALLING: Prevents the system from closing the camera app during calls.
  • TARGET_BUILD_FORCE_SYSTEM_TO_PLAY_MUSIC_WHILE_RECORDING: Allows music playback while recording video.
  • TARGET_BUILD_DISABLE_WIFI_CALLING: Disables Wi-Fi calling if set to true; otherwise, enables it if disabled.
  • TARGET_BUILD_SKIP_SETUP_JUNKS: Skips unnecessary setup steps.
  • BLOCK_NOTIFICATION_SOUNDS_DURING_PLAYBACK: Disables notification sounds while playing videos or audio files.
  • TARGET_BUILD_FORCE_SYSTEM_TO_PLAY_SMTH_WHILE_CALL: Allows the media player to play videos or songs during calls.
  • FORCE_ENABLE_POP_UP_PLAYER_ON_SVP: Enables the popup player in Samsung Video Player.
  • TARGET_BUILD_FORCE_DISABLE_SETUP_WIZARD: Disables the Setup Wizard. For enterprise or trusted use cases only. You are responsible for any legal consequences.
  • TARGET_FLOATING_FEATURE_INCLUDE_GAMELAUNCHER_IN_THE_HOMESCREEN: Adds Game Launcher to the home screen.
  • TARGET_FLOATING_FEATURE_BATTERY_SUPPORT_BSOH_SETTINGS: Adds battery health level settings similar to iPhone. Thanks to UN1CA.
  • BRINGUP_CN_SMARTMANAGER_DEVICE: Adds the Chinese version of Smart Manager and Device Care. Thanks to @saadelasfur. (Load Chinese applications at your own risk!) [Only works on UI6 and UI7]
  • TARGET_BUILD_ADD_SCREENRESOLUTION_CHANGER: Adds a screen resolution switcher for devices with 2340x1080 resolution. Thanks to @Yanndroid for their project.

Language & Locale

  • SWITCH_DEFAULT_LANGUAGE_ON_PRODUCT_BUILD: Sets the default language and region for the first boot. Set NEW_DEFAULT_LANGUAGE_ON_PRODUCT and NEW_DEFAULT_LANGUAGE_COUNTRY_ON_PRODUCT to your preferred values.

Miscellaneous

  • TARGET_DISABLE_SAMSUNG_ASKS_SIGNATURE_VERFICATION: Disables Samsung's ASKS signature verification. This does not affect the built-in signature verification and is only used for Samsung applications.
  • TARGET_ADD_ROUNDED_CORNERS_TO_THE_PIP_WINDOWS: Adds rounded corners to the PiP window. Tap to see an example.

Advanced

  • TARGET_INCLUDE_HORIZON_OEMCRYPTO_DISABLER: Removes DRM-related files. This will downgrade Widevine to L3 and disable HD playback. DRM apps might still work, but this is not guaranteed.
  • TINKER_MAX_REFRESH_RATE: Set to true and configure the variables to change your device’s maximum refresh rate. Thanks to @BrotherBoard.
  • DTBO_IMAGE_PATH: Path to your DTBO image.
  • THIS_IS_MY_DEVICE_MAX_REFRESH_RATE: Desired maximum refresh rate for your device.
  • MY_KEYSTORE_ALIAS: Enter your Keystore Alias.
  • MY_KEYSTORE_PASSWORD: Enter your Keystore Alias Password.
  • MY_KEYSTORE_PATH: Enter the path to your Keystore.
  • TARGET_REMOVE_SMARTSWITCH_DAEMON: Removes the Smart Switch listener port. It’s unnecessary, so removal is recommended.
  • TARGET_BUILD_REMOVE_SYSTEM_LOGGING: Removes unnecessary logging processes.
  • TARGET_BUILD_ADD_NETWORK_SPEED_WIDGET: Adds a network speed meter to the SystemUI quick settings.
  • TARGET_BUILD_BRING_NEWGEN_ASSISTANT: Brings Assistant 2.0 to Android ≤ 11.
  • RECOVERY_IMAGE_PATH: Set this if you want to patch your recovery image.
  • TARGET_INCLUDE_HORIZONUX_ELLEN: Ellen is a script that runs during boot to spoof or apply miscellaneous tweaks (does not alter performance).
  • TARGET_BUILD_MAKE_DEODEXED_ROM: Deodexes the ROM. Learn about ODEX and DEODEX.
  • TARGET_BUILD_FIX_ANDROID_SYSTEM_DEVICE_WARNING: Removes the post-boot Android System warning.
  • TARGET_DISABLE_FILE_BASED_ENCRYPTION: Disables FBE (file-based encryption) on internal storage. This was present in the script but undocumented until now.
