# To compile for arm64:
```
cd HorizonUX/src
make loader COMP_DEF="arm64"
```

# To compile for other arch (gcc should be present):
```
cd HorizonUX/src
make loader
```

# Build Errors:
- ignore the non-critical ones and drop a comment if it's critical
